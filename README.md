## 一、三分钟上手

### 1.1 安装fws
```
# 克隆项目
git clone https://github.com/sbfkcel/fws.git

# 进入fws目录
cd fws

# 安装依赖
npm install

# 注册工具
npm link

# 检查是否安装成功
fws -V
```

### 1.2 创建项目
```
fws create <项目名称>
```

### 1.3 监听项目

```
cd <项目目录>

fws watch
```
