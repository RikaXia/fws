```
node项目
=================================================

#期望的最终目录可能会是这样的
dev
    |- lib
    |   |- module1.js
    |   -- module2.js
    |- page
    |   |- homde.js
    |   -- pay.js
    |- index.js
    |- bin.js
    -- package.json

src
    |- ???



react项目
=================================================

#方案一
src
    |- tsx
    |   -- index.tsx
    -- jsx
        -- index.jsx        #由tsx/index.tsx编译出来，该文件最终会再编译到dev/js/index.js

#方案二
src
    -- tsx
        |- index.tsx
        -- index.jsx        #编译在tsx同级目录，该文件最终会再编译到dev/js/index.js

#期望的最终目录可能会是这样的
dev
    -- js
        -- index.js


=================================================
问题总结：node那样不能确定目录的不灵活以外，react、vue...文件关系对应更复杂，如果每个类型的项目多一种关系，整个东西下来就够乱了。
```
